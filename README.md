# hlongc Homebrew Tap

Homebrew tap for hlongc CLI tools.

## Install Claude Code Sentinel

```sh
brew tap hlongc/tap
brew install claude-code-sentinel
claude-code-sentinel install-managed
```

Verify the installation:

```sh
claude-code-sentinel doctor
```

Project: [hlongc/claude-code-sentinel](https://github.com/hlongc/claude-code-sentinel)
